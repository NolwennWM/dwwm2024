Les bases de SASS et SCSS

  • Pour installer SASS on utilisera
    npm install -g sass
  • Pour compiler du sass/scss en css on utilisera la commande
    sass cheminVerFichierSource.scss cheminVersFichierCible.sass
  • Pour éviter de relancer la commande à chaque modification, on peut y ajouter --watch
    sass --watch cheminVerFichierSource.scss cheminVersFichierCible.sass
    par exemple ici : sass --watch ./scss/test.scss ./css/style.css
  • On peut surveiller tout un dossier :
    sass --watch nomDossierSource/:nomDossierCible/
  • On peut choisir de minimifier notre compilation avec l'option "--style compressed"
    sass --watch nomDossierSource/:nomDossierCible/ --style compressed
